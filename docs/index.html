Your PDFs never leave your Mac

  • ✓No network calls, no analytics anywhere in the codebase.
  • ✓No uploads, no accounts. You choose files with the system dialog and save results back to disk.
  • ✓The activity log is a local file — ~/pdf-utils.log. Nothing is sent anywhere.
  • ✓Passwords are never stored or transmitted — encryption happens locally.
