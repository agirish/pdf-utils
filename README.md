# pdf-utils

A utility for managing PDF files.

## Capabilities

- **Add or remove pages** — Insert pages into a PDF or delete specific pages while keeping the rest intact.
- **Compress** — Reduce PDF file size for sharing or storage.
- **More** — Additional PDF operations can be added as the project grows.

## Status

This repository is set up as the home for these tools. Implementation details and usage will be documented here as features are added.
